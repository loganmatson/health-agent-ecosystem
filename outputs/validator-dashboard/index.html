⚠️ Sample data below is fictionalized for demonstration purposes. Real data populates automatically on each monthly Whoop upload via /optimize-week.

Why This Matters

Most AI health systems make predictions but never check if they were right. This validator closes that loop:

  • Neurologist Agent forecasts seizure risk — but does an elevated-stress day actually tank HRV the next morning? Now we know.
  • Gordon Ramsay selects meals by protein tier — but does chicken breast actually reduce next-day stress vs. ground beef? Now we track it.
  • Calendar Agent plans strain targets — but are PEAK sessions actually hitting ≥18 Whoop strain? Now we flag drift.
  • validator-history.json accumulates every monthly run — building a longitudinal dataset that shows whether these agents are genuinely getting smarter over time.
